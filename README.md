# ERP Sales & Delivery Module - Manual Testing Project

## Project Overview

This repository contains manual testing documentation for an ERP-based **Sales and Delivery module**.  
The purpose of this project is to demonstrate how I analyze business workflows, prepare test scenarios, write detailed test cases, execute tests, report bugs, and track issues in a structured QA process.

The module covers the full sales and delivery flow, starting from sales quotation creation to sales order processing, production order handling, delivery challan generation, and sales invoice completion.

---

## Application Under Test

The tested ERP module includes the following major sections:

- Product Portfolio
- Sales Quotations
- Sales Orders
- Production Orders
- Delivery Challans
- Sales Invoices
- Reports
- Permission-based workflow actions
